## Tutorial
To add an admin user, click the **Admin** dropdown in the top bar and select **Admins**. On this page, you can add a new admin user by clicking the **New Admin** button in the top-right corner. Only admin users created through the Frontiers application can be deleted within Frontiers.

To add an instructor user, click the **Admin** dropdown in the top bar and select **Instructors**. On this page, you can add a new admin user by clicking the **New Instructor** button at the top-right corner.

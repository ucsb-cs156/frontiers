## Tutorial
To create a course, click the **Create Course** button at the top-right corner of your instructor courses table. 

A modal will appear prompting you to enter the course name, term, and school. Fill out the form and submit it to create your course.

After submitting, the new course will appear in your instructor courses table. You can update the course name, term, or school using the **edit** button.

After creating a course, you can link a GitHub organization to your course by clicking the **Install GitHub App** button in that courseâ€™s row.

After clicking the button, you'll be prompted to choose an organization you want to link. 

Once the organization is linked, the **Install GitHub App** button will bew ill be replaced with a link to the GitHub organization and a Github settings icon for managing the connection between your course and GitHub.

To uninstall the Frontiers connection from the GitHub organization, click the GitHub settings icon. 

Then, on GitHub, scroll to the bottom of the page and click the **Uninstall**.
